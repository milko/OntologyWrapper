# getCollections( )

Return collection names

Return collection names

In this class we use the \MongoDB::listCollections() method, we extract only the name to conform with the method prototype: one should always instantiate an object derived from OntologyWrapper\CollectionObject when dealing wit collections.

This method will return the following retults:

  • FALSE: The database is not connected.
  • array: The database collection names.

Returns

array
List of collection names.
