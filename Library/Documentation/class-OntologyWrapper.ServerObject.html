Class ServerObject

Server object

This abstract class is the ancestor of all classes representing server connection instances, this class extends the OntologyWrapper\ConnectionObject class to implement server specific functionality prototypes.

# getStatistics( )

Return statistics

Return statistics

This method should return the server statistics, the result depends on the specific driver.

The method should return the following retults:

  • NULL: The operation is not supported.
  • FALSE: The server is not connected.
  • array: The server statistics.

We implement the method in this class as a fall-back.

Returns

array
Server statistics, NULL or FALSE.
