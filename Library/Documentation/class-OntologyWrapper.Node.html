Class Node

Node

This class implements a persistent OntologyWrapper\NodeObject instance, the class concentrates on implementing all the necessary elements to ensure persistence to instances of this class and referential integrity.

The object is considered initialised, OntologyWrapper\Node::isInited(), if it has at least the term reference, kTAG_TERM, or the tag reference, kTAG_TAG.

# ResolveObject( OntologyWrapper\ConnectionObject $theConnection, mixed $theIdentifier, boolean $doAssert = TRUE )

Resolve object

Resolve object

This method can be used to statically instantiate an object from the provided data store, it will attempt to select the object matching the provided native identifier and return an instance of the originally committed class.

The method accepts the following parameters:

  • $theContainer: The database or collection from which the object is to be retrieved.
  • $theIdentifier: The objet native identifier.
  • $doAssert: If TRUE, if the object is not matched, the method will raise an exception; if FALSE, the method will return NULL.

We implement this method to match objects in the nodes collection.

# preCommit( bitfield $theOperation = 0x00 )

Prepare object for commit

Prepare object for commit

In this class we first check if the object is OntologyWrapper\Node::isInited(), if that is not the case, we raise an exception, since the object cannot be committed if not initialised.

We then set the native identifier with a sequence number, if not yet set.

When deleting we check whether the object has its native identifier.
