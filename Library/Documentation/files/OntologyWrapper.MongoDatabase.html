getCollections() : array

Return collection names

In this class we use the {@link \MongoDB::listCollections()} method, we extract only the name to conform with the method prototype: one should always instantiate an object derived from {@link CollectionObject} when dealing wit collections. This method will return the following retults:
  • FALSE: The database is not connected.
  • array: The database collection names.
