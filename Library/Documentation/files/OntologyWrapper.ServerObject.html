getStatistics() : array

Return statistics

This method should return the server statistics, the result depends on the specific driver. The method should return the following retults:
  • NULL: The operation is not supported.
  • FALSE: The server is not connected.
  • array: The server statistics.
We implement the method in this class as a fall-back.

Return a new database instance

This method should implemented by concrete derived classes, it expects a list of offsets which include server information and should use them to instantiate a {@link DatabaseObject} instance. Derived classes must implement this method. When implementing this method you should not forget to set the dictionary.
