TagCacheObject

Tag cache object

This abstract class is the prototype of classes that implement a tag cache.

The main purpose of a tag cache is to hold in a fast retrieval cache all the current ontology {@link TagObject} objects and information, this cache is actively used by all classes derived from {@link OntologyObject} for resolving offsets into {@link TagObject} references and to cast offset values according to the {@link TagObject} declared data type.

Concrete derived classes should implement a constructor that will instantiate the cache and load all current tag identifiers and objects.

The class features the following specific virtual methods:

  • {@link init()}: This method should initialise the cache.
  • {@link setTagId()}: This method should set a key/value pair in which the key represents the tag global identifier and the value the tag native identifier.
  • {@link getTagId()}: This method should return a tag native identifier given a tag global identifier.
  • {@link delTagId()}: This method should delete a key/value pair in which the key represents the tag global identifier.
  • {@link setTagObject()}: This method should set a key/value pair in which the key represents the tag native identifier and the value the tag object or array of contents.
  • {@link getTagObject()}: This method should return a tag object or array of contents given a tag native identifier.
  • {@link delTagObject()}: This method should delete a key/value pair in which the key represents the tag native identifier.
  • {@link getTagGID()}: This method should return a tag global identifier given a tag native identifier.

Set a tag identifier

setTagId(string $theKey, integer $theValue)
abstract

This method should set a key/value pair in the cache, in which the key represents a tag global identifier and the value the tag native identifier.

The global identifier will be cast to string and the native identifier will be cast to integer.

The method should raise an exception if the operation fails.

Get a tag global identifier

getTagGID(integer $theKey, boolean $doAssert = FALSE) : mixed

This method should return a tag global identifier given a tag native identifier.

If the provided native identifier is not matched and the second parameter is

TRUE, the method should raise an exception; if the parameter is FALSE, the method should return NULL. If you provide {@link kTAG_NID}, the method will return it. Note that the provided key will be cast to integer, which means that if you did not provide a numeric value, the results will be unexpected.
