Detailed Description

Node

This class implements a persistent NodeObject instance, the class concentrates on implementing all the necessary elements to ensure persistence to instances of this class and referential integrity.

The object is considered initialised, isInited(), if it has at least the term reference, kTAG_TERM, or the tag reference, kTAG_TAG.

OntologyWrapper\Node::loadTag ( )

Load tag object

This method can be used to resolve the tag into an object.

The method will return the tag object if the operation succeeded and NULL if the object is not committed, if the object does not hold a collection reference, or if the object has no tag.

If the tag cannot be resolved, the method will raise an exception.

We then set the native identifier with a sequence number, if not yet set.

Resolve object

This method can be used to statically instantiate an object from the provided data store, it will attempt to select the object matching the provided native identifier and return an instance of the originally committed class.

The method accepts the following parameters:

  • $theContainer: The database or collection from which the object is to be retrieved.
  • $theIdentifier: The objet native identifier.
  • $doAssert: If TRUE, if the object is not matched, the method will raise an exception; if FALSE, the method will return NULL.

We implement this method to match objects in the nodes collection.
