Detailed Description

Tag cache

This class is a concrete derived instance of the TagCacheObject which implements a tag cache that uses the Memcached class.

The class implements the following additional methods:

  • init(): The method will flush the current cache and reload all tags.
  • Pid(): The method will return the connection persistent identifier.
  • Connection(): The method will return the connection resource, which is, in this case, a Memcached instance.
  • stats(): The method will return the connection statistics.

This class may be used as a template to implement tag caches that use other cache engines if the Memcached class is not available.

OntologyWrapper\TagCache::__construct (   $theIdentifier,

Instantiate class.

The constructor accepts two parameters:

  • $theIdentifier: This string parameter represents the { Memcached} persistent identifier provided to its constructor, the parameter is required, since the cache should be active across sessions.
  • $theServers: This array parameter represents the list of servers that serve the cache, it is equivalent to the parameter of the Memcached::addServers() method, it is a list of elements comprised by three parameters:
    • Host: The server host.
    • Port: The server port.
    • Weight: The weight of the server relative to the total weight of all the servers in the pool.
    This parameter may be omitted if the cache has been initialised beforehand.

The constructor will first instantiate the Memcached object, then it will check if the connection resource has already a list of servers associated, if that is the case we assume the cache is already initialised; if that is not the case, we will add the servers provided in the second parameter and load all the current tag identifiers and objects.

This method will set a key/value pair in the cache consisting of the tag global identifier as the key and the tag native identifier as the value; both values will be cast.

The expiration period is infinite by default, since the tags should persist in the cache.
