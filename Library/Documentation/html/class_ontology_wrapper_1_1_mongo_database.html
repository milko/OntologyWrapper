Detailed Description

Mongo database

This class is a concrete implementation of the DatabaseObject wrapping a MongoDB class.

OntologyWrapper\MongoDatabase::getSequenceNumber (   $theSequence)

Return sequence number

In this class we match the provided parameter string with an entry in the kSEQ_COLLECTION collection in the database, the native identifier of the record is the sequence selector, while the sequence number will be found at the kSEQ_OFFSET offset.
