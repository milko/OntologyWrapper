Detailed Description

Tag

This class implements a persistent TagObject instance, the class concentrates on implementing all the necessary elements to ensure persistence to instances of this class and referential integrity.

The object is considered initialised, isInited(), if it has at least the terms path, kTAG_TERMS, with an odd number of elements, the data type, kTAG_DATA_TYPE, and the label, kTAG_LABEL.

OntologyWrapper\Tag::preCommit (   $theOperation = 0x00)
protected

Prepare object for commit

In this class we first check if the object is isInited(), if that is not the case, we raise an exception, since the object cannot be committed if not initialised.

We then set the native identifier, if not yet filled, with the global identifier generated by the __toString() method.

We finally set the sequence number, kTAG_SEQ, if it is not yet set by requesting it from the database of the current object's container.

When deleting we check whether the object has its native identifier.

Resolve object

This method can be used to statically instantiate an object from the provided data store, it will attempt to select the object matching the provided native identifier and return an instance of the originally committed class.

The method accepts the following parameters:

  • $theContainer: The database or collection from which the object is to be retrieved.
  • $theIdentifier: The objet native identifier or sequence number.
  • $doAssert: If TRUE, if the object is not matched, the method will raise an exception; if FALSE, the method will return NULL.

We implement this method to match objects in the tags collection by matching string identifiers with the native identifier and integer identifiers with the sequence number.
