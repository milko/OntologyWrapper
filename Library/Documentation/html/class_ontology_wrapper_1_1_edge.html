Detailed Description

Edge

This class implements a directed graph by relating a subject vertex with an object vertex through a predicate, the direction of the relationship is from the subject to the object.

The vertices of this relatonship, the subject and object, are Node instance references, while the relationship predicate is represented by a Term instance reference.

The class features the following default offsets:

  • kTAG_NID: Native identifier. This required attribute holds a string which represents the combination of the subject, predicate and object of the relationship. This attribute must be managed with its offset, although in derived classes it will be set automatically.
  • kTAG_SUBJECT: Subject. This attribute represents the origin of the relationship, it is an integer value representing the reference to a Node instance. This attribute must be managed with its offset.
  • kTAG_PREDICATE: Predicate. This attribute represents the type of relationship, it is a string value representing the reference to a Term instance. This attribute must be managed with its offset.
  • kTAG_OBJECT: Object. This attribute represents the destination of the relationship, it is an integer value representing the reference to a Node instance. This attribute must be managed with its offset.

The __toString() method will return the value stored in the native identifier, if set, or the computed native identifier, which is the concatenation of the subject, predicate and object references separated by the kTOKEN_INDEX_SEPARATOR token.

Objects of this class feature a primary key which is not persistent: the vertices referenced in the native identifier are integer sequences which depend on the order these objects were inserted: this means that both Node and Edge instances must be re-created when exported.

Objects of this class can hold any additional attribute that is considered necessary or useful to define and share the current node. In this class we define only those attributes that constitute the core functionality of the object, derived classes will add attributes specific to the domain in which the object will operate.

The object is considered initialised, isInited(), if it has at least the subject, predicate and object references.
