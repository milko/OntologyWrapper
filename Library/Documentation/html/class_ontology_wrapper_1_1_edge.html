Detailed Description

Edge

This class implements a persistent EdgeObject instance, the class concentrates on implementing all the necessary elements to ensure persistence to instances of this class and referential integrity.

The object is considered initialised, isInited(), if it has at least the subject, predicate and object references.

OntologyWrapper\Edge::loadObject ( )

Load object vertex object

This method can be used to resolve the object vertex into an object.

The method will return the object vertex object if the operation succeeded and NULL if the object is not committed, if the object does not hold a collection reference, or if the object has no object vertex.

If the object vertex cannot be resolved, the method will raise an exception.

OntologyWrapper\Edge::preCommit (   $theOperation = 0x00)
protected

Prepare object for commit

In this class we first check if the object is isInited(), if that is not the case, we raise an exception, since the object cannot be committed if not initialised.

We then set the native identifier, if not yet filled, with the global identifier generated by the __toString() method.

When deleting we check whether the object has its native identifier.

static

Resolve object

This method can be used to statically instantiate an object from the provided data store, it will attempt to select the object matching the provided native identifier or the provided array of subject, predicate, object references and return an instance of the originally committed class.

The method accepts the following parameters:

  • $theContainer: The database or collection from which the object is to be retrieved.
  • $theIdentifier: The objet native identifier.
  • $doAssert: If TRUE, if the object is not matched, the method will raise an exception; if FALSE, the method will return NULL.

We implement this method to match objects in the edges collection.
