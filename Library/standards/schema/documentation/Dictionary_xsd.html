  </xs:annotation>
  <xs:complexType>
    <xs:sequence>
      <xs:annotation>
        <xs:documentation xml:lang="en">Edges require three elements: the subject, the predicate and the object. The subject and the object are node references, while the predicate is a term reference. The elements must have the tag constant or persistent identifier attribute and the value of the element must be either the node or the term persistent identifier. If either the subject or the object is omitted, it is assumed that the node defined in the current unit block is implied; in this case there can only be one node in the block. If the predicate is omitted, it is assumed that the term defined in the current block is implied; in this case there can only be one term in the block.</xs:documentation>
